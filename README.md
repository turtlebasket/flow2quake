# Flow2quake

## Environment Setup

Requires:

- [Anaconda](https://www.anaconda.com/download/) or [Miniconda](https://docs.anaconda.com/miniconda/)

Create Conda environment from file (currently Linux-only):
```
conda create -f environment-ubuntu.yml -n flow2quake
```
